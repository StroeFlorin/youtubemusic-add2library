# YouTube Music - Add2Library

A small Chrome extension that adds an `Add to library` toggle button to the YouTube Music player bar.

## What it does

- Adds a quick button near the player controls on `music.youtube.com`
- Lets you save/remove the current track from your library
- Includes a popup toggle to enable or disable the feature

## Install (unpacked)

1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select this project folder

## Usage

1. Open [https://music.youtube.com](https://music.youtube.com)
2. Play a track
3. Use the new library button in the player bar, or the native 3-dot menu
4. Use the extension popup to enable/disable the button

## Permissions

- `tabs`: communicate with and reload the active YouTube Music tab
- `storage`: save extension settings
